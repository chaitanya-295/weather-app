# 🌤 Weather Forecast Web Application

A full-stack weather forecasting web application that provides real-time weather updates, forecasts, and interactive features using the OpenWeatherMap API.

---

## 🚀 Features

- 🌍 Real-time weather based on user location
- 🔍 Search weather by city name
- ⭐ Add/remove favorite cities
- 🕒 Recent search history (limited & optimized)
- 📊 Hourly temperature chart
- 🌧 Rain probability insights
- 🧠 AI-based weather summary
- 🗺 Interactive map with location selection
- 🌙 Light/Dark mode support
- 📱 Fully responsive design
- ✨ Smooth animations using Framer Motion

---

## ⚙️ Setup Instructions

🔹 Dependencies

    All required dependencies are listed in the respective `package.json` files.

🔹 Backend Setup

    cd backend

    npm install

Create .env file:

    API_KEY=your_openweather_api_key
    
    MONGO_URI=your_mongodb_connection_string

Run backend:

    node server.js

🔹 Frontend Setup

    cd frontend
    
    npm install

Run frontend:

    npm run dev
